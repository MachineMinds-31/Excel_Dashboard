# Excel_Dashboard
# 🚀 HSSE Violation Dashboard (Excel + Power Query)

## 📌 Overview
This project is an automated HSSE (Health, Safety, Security & Environment) dashboard built using Excel and Power Query.

It transforms raw violation logs into meaningful insights with dynamic calculations and visualizations.

---

## 🔧 Features

- 📂 Automated data ingestion from folder (Power Query)
- 🔄 One-click refresh for monthly updates
- 📊 Dynamic violation tracking (no pivot dependency)
- 🧠 Smart formulas using COUNTIFS (handles missing categories)
- 🏆 Best Performing GA identification
- 🚨 Highest Violation Contributor tracking
- 📉 Violations/Vehicle/Day KPI

---

## ⚙️ Tools Used

- Microsoft Excel
- Power Query
- Data Cleaning & Transformation
- Dashboard Design

## 📁 How to Use

1. Place new monthly files into the source folder
2. Open the Excel dashboard
3. Click **Refresh**
4. Dashboard updates automatically
